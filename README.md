## Featured Projects

### Ryanair Passenger Reviews Analysis

In a competitive aviation market, customer feedback is essential for service optimization. This project leverages Natural Language Processing (NLP) to analyze Ryanair passenger reviews spanning 2012–2024, employing sentiment analysis, topic modeling, and root cause identification.

The analysis identifies key satisfaction and dissatisfaction drivers, highlights operational challenges, and pinpoints routes with recurring issues. A custom machine learning sentiment classifier automates categorization, while graph network visualizations trace negative sentiment patterns to specific routes, delivering actionable insights for operational and customer service improvement.

### Amazon Product Bundling and Recommendation

This comprehensive analysis of Amazon sales data extracts valuable insights into sales trends, customer purchase behavior, and profitability drivers. Using data mining techniques—exploratory data analysis, clustering, association mining, and predictive modeling—the project identifies meaningful patterns to construct effective product bundling strategies and personalized customer recommendations.

### IntelliFraud: Bank Account Fraud Detection

IntelliFraud is an innovative platform designed to enhance detection of fraudulent bank account applications. The tool combines statistical exploration, dynamic graph network analysis, and advanced machine learning for comprehensive fraud detection.

Key capabilities include exploratory data analysis across customizable datasets, fraud network analysis for transaction flow visualization, model evaluation for classification metrics, and an inference interface with SHAP and ELI5 integration for model interpretability. These features empower organizations to significantly improve fraud detection effectiveness.
